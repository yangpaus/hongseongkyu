# 홍성규 경기도지사후보 공식 웹사이트
진보당 경기도지사 후보 기호 5번 홍성규의 공식 선거 웹사이트입니다.
홍성규 후보의 출마의 변과 공약을 소개하는 정적 HTML 사이트입니다.

```
├── index.html       # 메인 페이지
├── style.css        # 스타일시트
├── img/             # 이미지 파일
└── schedule/        # 후보 일정 관리 템플릿
```

## 실행 방법
별도의 빌드 과정 없이 `index.html`을 브라우저에서 열거나,
로컬 서버를 실행해서 확인할 수 있습니다.
